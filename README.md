# Lógica de Programación

Este repositorio contiene material utilizado en clases introductorias
de lógica de programación.

## Contenido

- Diapositivas de las clases
- Ejemplos de algoritmos
- Recursos para estudiantes principiantes

## Objetivo

Ayudar a estudiantes a comprender los fundamentos
del pensamiento lógico y la programación.
